# E-Commerce-Website#


🚀 TechPro | Advanced E-Commerce Portal
TechPro is a high-performance, responsive, Single Page Application (SPA) designed for modern e-commerce needs. This project integrates a dynamic shopping cart, product recommendation engine, and essential business sections (About, Contact, Privacy) into a seamless, unified experience.

🛠 Features
🛒 Dynamic Shopping Cart
Real-time Updates: Add, remove, and adjust item quantities with instant subtotal and tax calculations.

Persistence: Utilizes localStorage to ensure user cart data is saved even after page refreshes.

Empty State Handling: Intuitive UI transitions when the cart is empty to encourage further shopping.

📱 Fully Responsive UI
Mobile-First Design: Optimized for all screen sizes using CSS Flexbox and Grid.

Hamburger Menu: Smooth navigation toggle for mobile users.

Adaptive Layouts: Sections like the Contact Form and About Us stats automatically stack on smaller viewports.

⚡ Single Page Architecture (SPA)
Instant Switching: Navigate between Cart, About, Contact, and Privacy sections without a single page reload.

Animated Transitions: Smooth fade-in effects for improved User Experience (UX).

📧 Integrated Business Sections
About Us: Professional mission statement with a statistics counter grid.

Contact System: A functional layout featuring a contact info card and a validated inquiry form.

Privacy Policy: A clean, readable legal template with structured hierarchy.

🏗 Technology Stack
HTML5: Semantic structure for SEO and accessibility.

CSS3: Custom properties (variables), Grid, Flexbox, and Keyframe animations.

JavaScript (ES6+): Vanilla JS for DOM manipulation, state management, and SPA routing.

FontAwesome: High-quality vector icons for UI elements.

Google Fonts: "Inter" typeface for professional, clean typography.

🚀 Getting Started
Download: Save the provided code as an .html file (e.g., index.html).

Run: Open the file in any modern web browser (Chrome, Firefox, Edge, Safari).

Customize:

Change the colors in the :root section of the CSS.

Modify the cartItems or RECOM_DATA arrays in the <script> tag to add your own products.

📂 File Structure
The project is delivered as a Single-File Solution, making it extremely easy to deploy.
